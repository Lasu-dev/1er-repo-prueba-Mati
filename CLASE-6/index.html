Etiquetas semánticas

Le dan una estructura semántica a nuestra página, para poder contarle a los navegadores como está estructurada.
Siguen siendo "cajas" como los divs, pero con el plus de que tienen un significado, refieren a cierta sección de la página, son conceptuales y más descriptivas que los divs, y son mas fáciles de entender para los navegadores.

Header:

Encabezado o Cabezal, ya que puede contener algunos elementos de encabezado, así como también un logo, un menú de navegación, un título de la página, un formulario de búsqueda, etc. Representa un grupo de ayudas introductorias o de navegación.

Click para ir a la Documentación oficial acerca de la etiqueta "header"

Main

El main suele tener todo el contenido principal de la página, puede tener secciones para representar ese contenido, solo en casos especiales podría contener un div, es mucho mas apropiado que contenga secciones y dentro de cada section, se puede contener mas sections o articles. Esta etiqueta es muy importante, ya que es la que se usa para indicar que es el contenido principal de la página, y le vamos a poner un class = "hero" para que se pueda identificar en el CSS.

Aside

Representa el contenido de la página que está indirectamente relacionado con el contenido principal, este contenido también podría ser independiente.
